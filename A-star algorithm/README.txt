How to run program:
	1) Enter a threshold size between 1 - 100
	2) Enter a grid size, preferably 0.002 or 0.003
	3) enter the x coordinate of your starting point
	4) enter the y coordinate of your starting point
	5) enter the x coordinate of your end point
	6) enter the y coordinate of your end point
	7) The graph will be shown and a red line from the starting point to the ending point if and only if there is a path
